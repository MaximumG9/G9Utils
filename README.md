G9Utils
=======
A client side "utilities mod" including features
for visualization, bug finding, silly fun, and
things that would be considered by most to be 
cheats.

Compiling
=========
* Open a shell/command prompt
* Clone the repository with something like:
```
git clone "https://github.com/MaximumG9/G9Utils.git"
```
* In the project folder run `gradlew build` or `./gradlew build`
* The built jar file will be in `build/libs/`
